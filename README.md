# To_Do_List_FocusFlow

This is a basic full-stack Todo List application built using MongoDB, Express.js, React.js, and Node.js. The project is created to understand how frontend and backend communicate using REST APIs.

The frontend is built with React.js and uses Fetch API to send requests to the backend. The backend is built using Node.js and Express.js, while MongoDB is used to store todo data.

Users can add, update, delete, and mark tasks as completed. All data is stored in MongoDB, making it a complete CRUD-based MERN stack application.

Features:
- Add new todos
- View all todos
- Update todo status
- Delete todos
- REST API integration
- MongoDB database connectivity
- Responsive frontend UI
- search todos
- dark/light theme

Technologies Used:
- React.js
- vanilla
- Node.js
- Express.js
- MongoDB
- Mongoose
- Fetch API

API Endpoints:
- GET /api/todos
- POST /api/todos
- PUT /api/todos/:id
- DELETE /api/todos/:id

This project focuses on learning CRUD operations, API creation, frontend-backend integration, and database management using the MERN stack.
